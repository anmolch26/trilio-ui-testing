Shopify Integration
To enable API communication with your Shopify store:
Store URL:
e.g. https://your-store.myshopify.com
Access Token:
Private app access token
Required Admin API Scopes:
* read_products - Access product information
* read_product_listings - Access product listings
* read_orders - Access order data
* read_customers - Access customer information
* read_inventory - Access inventory levels
* read_analytics - Access analytics and reports
* read_reports - Access additional reporting data
* read_marketing_events - Access marketing event data
* read_fulfillments - Access fulfillment information
* read_shipping - Access shipping information
* read_discounts - Access discount codes and promotions
* read_price_rules - Access pricing rules
* read_gift_cards - Access gift card data
* read_shop_data - Access general shop information
* read_checkouts - Access checkout data
* read_abandoned_checkouts - Access abandoned cart data
* read_content - Access blog posts, pages, and other content
Process:
1. Log in to your Shopify Admin
2. Go to Settings > Apps and sales channels > Develop apps
3. Enable custom apps (if not already enabled)
4. Click Create an app
5. Configure Admin API access with all the scopes listed above
6. Install the app
7. Copy the Admin API access token
